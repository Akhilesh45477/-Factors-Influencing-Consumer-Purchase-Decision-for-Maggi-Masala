# Factors-Influencing-Consumer-Purchase-Decision-for-Maggi-Masala
This project examines factors influencing consumers’ decision to repurchase Maggi Masala using survey-based primary data. Factor Analysis (PCA with Varimax rotation) reduced 29 variables into six key factors: Brand, Taste, Emotional Connect, Affordability, Availability, and Advertisement. Regression Analysis showed these factors significantly impact repurchase intention (R² ≈ 44%), providing insights to improve marketing and sales strategies.
