# Factors-Influencing-Consumer-Purchase-Decision-for-Maggi-Masala
This project analyzes consumer preferences and purchasing behavior toward Maggi Masala using statistical techniques. Primary data was collected through a structured questionnaire based on a 1–5 Likert scale covering multiple perception variables.
#-Objective
-Identify key factors influencing customer repurchase decisions.
-Reduce multiple survey variables into meaningful dimensions.
-Study how these factors impact the likelihood of buying and recommending Maggi Masala.
Factor Analysis (PCA with Varimax rotation) reduced 29 variables into six key factors: Brand, Taste, Emotional Connect, Affordability, Availability, and Advertisement. Regression Analysis showed these factors significantly impact repurchase intention (R² ≈ 44%), providing insights to improve marketing and sales strategies.
